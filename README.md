# Power BI Sales & Business Performance Dashboard

##  Project Overview
This project focuses on analyzing sales and business performance using Power BI.
The dashboard provides actionable insights into sales trends, profit, and customer behavior.

##  Tools Used
- Power BI
- Excel
- DAX
- Power Query

##  Key Features
- Interactive dashboards with slicers and filters
- KPI tracking (Total Sales, Profit, Growth %)
- Data cleaning and transformation using Power Query
- DAX measures for business calculations
- Drill-down and trend analysis

##  Dataset
Sales data stored in Excel format and transformed in Power BI.
